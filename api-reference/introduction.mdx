## Authentication Overview

The API supports authentication via API Keys (using the `X-API-Key` header) or session Cookies. 
Refer to the details within each endpoint for specific security scheme requirements.

For example, the general security definitions in the OpenAPI spec are:
```json
"securitySchemes": {
  "ApiKeyAuth": {
    "type": "apiKey",
    "in": "header",
    "name": "X-API-Key"
  },
  "CookieAuth": {
    "type": "apiKey",
    "in": "header",
    "name": "Cookie"
  }
}
```
And individual endpoints will reference these schemes.
